First Module

Front End Development

Front-end engineers work closely with designers to make websites beautiful, functional, and fast. This Career Path will teach you not only the necessary languages and technologies, but how to think like a front-end engineer, too. By the end, you’ll have the portfolio and interview skills you need to start for your new career.

  • Basics , HTML/CSS3 (layout, Typography, Responsive Design, Animation]
  • Workshop (Loading Animation Notification Bars, CSS3 Tabs, Parallax Section, Motion Typography, Power Point like presentation)
  • HTML5 (Geolocation, Local Storage, Web Storage, Canvas, WebGI).
  • .Bootstrap (Car Sales Website, Daraz, Stylo Shoes, Facebook type Website).
  • .JavaScript(Dom, Event, Ajax, Templating)
  • .React/Redux(Quiz App, To-dos App, Twitter App, Facebook Clone)
  • .Angular8 MEAN (Mean To-dos, My Weather, Movie Finder etc.)

Second Module

Back End Development

No matter what you’re building, having a strong foundation is crucial — that’s where back-end engineers come in. In this Career Path, you’ll start with programming servers and client-side interfaces, then level up to designing databases. By the end, you’ll have everything you need to start your career as a back-end engineer.

  • .Node.js/ Mongo DB (CURD App. Node Blog System E-Commerce App)
  • .Next.js /Redis (Code Splitting, Better SEO, performance etc)
  • .GraphQL(Social Media App, Integration of Apollo Client with React.js)
  • .Typescript (Express React Website, Board Application, etc.)
  • .C#6 (payroll system, Media player, Game, Security login system, etc.)
  • .Asp.net and SqlServer 2014(Dash Board Application and Data Entry System).
  • Asp.net MVC6 and Bootstrap.

Third Module

Mobile Apps Development

In this course, you’ll be introduced to the wonderful world of Android. You’ll find out why Android is such an awesome place to be developing apps. You’ll meet Android Studio—the Integrated Development Environment (IDE) for Android, and you’ll learn the fundamentals skills you need to build simple Android apps and run them on a real or simulated virtual device!

  • React Native(IOS Apps, Creating Calculator App, Tab Navigation, Barcode Scanner)
  • Full Ecommerce Project using React Native & AWS Amplify.
  • Flutter

Fourth Module

Development and Operations

DevOps is an application software development methodology. In order to streamline the process, DevOps brings together developers and the operations team. This helps drive productivity and collaboration by automating infrastructure and workflows as well as evaluating performance.

  • Dockers (Container Introduction, Installation, Custom Images, Docker Compose)
  • .Google Cloud (Authorization, Cloud Storage Overview, Instance Groups, Cloud SQL, Autoscaling, load balancing)
  • Kubernetes
  • .AWS Cloud Architecture
  • .AWS LAMDA AWS Storage(s3)
  • .AWS SES
  • .Microsoft Azure (Azure portals, Management tools)
  • .Digital Ocean
